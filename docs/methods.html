Methodology

Data sources, definitions, and analysis approach.

Definition of Father Absence

  • Binary indicator: did not live continuously with biological father from birth through age 17.
  • Coded from NLSY97 retrospective household composition roster.
  • 8 mutually exclusive categories: always-resident, absent-from-birth, early exit (0–5), middle exit (6–11), adolescent exit (12–17), intermittent, deceased, other/unknown.
  • Primary analysis uses binary (any absence vs continuous presence).

Identification Hierarchy

Tier Approach Status
Descriptive Survey-weighted means, group contrasts Complete
Adjusted association OLS/GLM with pre-exposure covariates Complete
Quasi-causal Event-time post-only, sibling FE, IPW Event-time complete; sibling FE insufficient
Exploratory ML Elastic net, random forest, feature importance Complete

Pre-Exposure Covariates

  • Child sex
  • Race/ethnicity
  • Maternal age
  • Maternal education
  • Baseline household income
  • Birth order
  • Sibship size
  • Region
  • Maternal employment
  • Maternal cognitive proxies (AFQT)

Limitations

These limitations should be considered when interpreting any results from this analysis.

  • NLSY97 is unweighted; cohort-specific (born 1980–84).
  • Father absence is association, not proven causal.
  • ACS uses ESP-based proxy (current household), not full childhood history.
  • Sibling FE design lacks sufficient within-family variation.
  • Single-jurisdiction parole data for event-time design.
